## 摘要
大型基础模型为自主系统提供了强大的推理能力，但将语义意图映射到可靠的实时控制仍具挑战。现有方法要么 (i) 让大语言模型 (LLM) 直接生成轨迹——这种方式脆弱、难以验证且易产生延迟；要么 (ii) 在线调整模型预测控制 (MPC) 的目标——这混合了慢速深思熟虑与快速控制，模糊了接口界限。我们提出了智能体快慢规划（Agentic Fast-Slow Planning），这是一个在自然时间尺度上解耦感知、推理、规划和控制的分层框架。该框架包含两个桥梁：Perception2Decision 使用车载视觉语言模型 (VLM) 检测器将场景压缩为以自我为中心的拓扑结构，然后通过云端 LLM 决策器将其映射为符号化驾驶指令，在保持可解释性的同时减少了带宽和延迟；Decision2Trajectory 则将指令转换为可执行路径，其中语义引导的 A* 算法（Semantic-Guided A*）将源自语言的软成本嵌入经典搜索中，使解偏向可行轨迹，而智能体细化模块（Agentic Refinement Module）则利用反馈和记忆来调整规划器的超参数。最后，MPC 实时跟踪轨迹，并针对困难情况提供可选的云端引导参考。在 CARLA 中的实验表明，与纯 MPC 和 A* 引导的 MPC 基准相比，智能体快慢规划提高了扰动下的鲁棒性，横向偏差减少了高达 45%，完成时间缩短了 12% 以上。代码可在 https://github.com/cjychenjiayi/icra2026_AFSP 获取。

## Abstract
Large foundation models enable powerful reasoning for autonomous systems, but mapping semantic intent to reliable real-time control remains challenging. Existing approaches either (i) let Large Language Models (LLMs) generate trajectories directly - brittle, hard to verify, and latency-prone - or (ii) adjust Model Predictive Control (MPC) objectives online - mixing slow deliberation with fast control and blurring interfaces. We propose Agentic Fast-Slow Planning, a hierarchical framework that decouples perception, reasoning, planning, and control across natural timescales. The framework contains two bridges. Perception2Decision compresses scenes into ego-centric topologies using an on-vehicle Vision-Language Model (VLM) detector, then maps them to symbolic driving directives in the cloud with an LLM decision maker - reducing bandwidth and delay while preserving interpretability. Decision2Trajectory converts directives into executable paths: Semantic-Guided A* embeds language-derived soft costs into classical search to bias solutions toward feasible trajectories, while an Agentic Refinement Module adapts planner hyperparameters using feedback and memory. Finally, MPC tracks the trajectories in real time, with optional cloud-guided references for difficult cases. Experiments in CARLA show that Agentic Fast-Slow Planning improves robustness under perturbations, reducing lateral deviation by up to 45% and completion time by over 12% compared to pure MPC and an A*-guided MPC baseline. Code is available at https://github.com/cjychenjiayi/icra2026_AFSP.

#### 1. 核心问题与整体含义（研究动机和背景）
自动驾驶系统正逐渐引入大语言模型（LLM）和视觉语言模型（VLM）作为推理引擎，以应对复杂场景。然而，**如何将高层语义意图转化为可靠的实时控制**仍是核心挑战。
*   **现有痛点**：直接让 LLM 生成轨迹（如 Trajectory-LLM）往往过于脆弱且难以验证；而通过语言调整 MPC 参数的方法则模糊了“慢速决策”与“快速执行”的界限，导致接口混乱。
*   **研究目标**：提出 **Agentic Fast-Slow Planning (AFSP)** 框架，借鉴人类认知的双重过程理论，在不同时间尺度上解耦感知、推理、规划与控制，实现大模型推理能力与经典控制稳定性的有机结合。

#### 2. 方法论：核心思想与关键技术
AFSP 框架通过两个关键“桥梁”连接感知与执行：

*   **Perception2Decision (感知到决策)**：
    *   **车载端**：使用轻量化 VLM（如 InternVL-2B）作为拓扑检测器，将原始图像压缩为以自我为中心的拓扑图（包含物体类别、距离、方位等），极大降低了上传云端的带宽需求。
    *   **云端**：LLM 接收拓扑图，通过思维链（CoT）和上下文学习生成符号化驾驶指令（如 `LEFT`, `RIGHT`, `KEEP`）及驾驶风格。
*   **Decision2Trajectory (决策到轨迹)**：
    *   **Semantic-Guided A\***：在经典 A* 算法的代价函数 $f(s) = g(s) + h(s)$ 中引入**语义软成本 $c_{sem}$**。根据 LLM 指令的完成情况（正确、延迟、错误、过度执行）分配不同的惩罚值，引导搜索算法生成符合语义意图且几何可行的路径。
    *   **Agentic Refinement Module**：利用 LLM 智能体自动调整规划器的超参数（如各项成本权重）。它通过观察规划反馈和记忆历史成功案例，替代人工调参过程。
*   **实时控制**：使用 **Cloud-Guided MPC**。在常规情况下使用本地参考轨迹，在复杂或困难场景下切换至云端引导的全局参考轨迹，确保实时性与安全性。

#### 7. 优点（亮点）
*   **尺度感知解耦**：科学地将“慢速语义推理”与“快速几何规划/控制”分离，解决了 LLM 直接控制的不可靠问题。
*   **语义软成本设计**：首次提出将语言指令转化为 A* 搜索的软约束，既保留了经典算法的确定性，又引入了 LLM 的灵活性。
*   **智能体闭环调参**：通过 Agent 自动优化规划器参数，提升了系统在长尾场景下的自适应能力。

#### 8. 不足与局限
*   **仿真依赖**：所有实验均在 CARLA 仿真环境中完成，尚未在真实道路或复杂天气/光照条件下进行实车验证。

*   **计算开销**：虽然 VLM 是轻量级的，但在嵌入式车载平台上运行 2B 规模的模型仍需考虑实时功耗与计算资源的分配。
