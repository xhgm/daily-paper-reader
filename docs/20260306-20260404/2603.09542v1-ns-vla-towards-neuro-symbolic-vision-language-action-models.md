## 摘要
视觉-语言-动作（VLA）模型旨在将指令落地于视觉语境中，并为机器人操纵生成动作序列。尽管近期取得了进展，VLA 模型在学习相关且可重用的原语、减少对大规模数据和复杂架构的依赖，以及实现超越演示的探索方面仍面临挑战。为了应对这些挑战，我们提出了一种通过在线强化学习（RL）实现的创新神经符号视觉-语言-动作（NS-VLA）框架。该框架引入了一个符号编码器来嵌入视觉和语言特征并提取结构化原语，利用符号求解器实现高数据效率的动作序列化，并利用在线强化学习通过扩展性探索来优化生成。在机器人操纵基准测试上的实验表明，NS-VLA 在单样本训练和数据扰动设置下均优于以往的方法，同时展现出卓越的零样本泛化能力、高数据效率和扩展的探索空间。我们的代码已开源。

## Abstract
Vision-Language-Action (VLA) models are formulated to ground instructions in visual context and generate action sequences for robotic manipulation. Despite recent progress, VLA models still face challenges in learning related and reusable primitives, reducing reliance on large-scale data and complex architectures, and enabling exploration beyond demonstrations. To address these challenges, we propose a novel Neuro-Symbolic Vision-Language-Action (NS-VLA) framework via online reinforcement learning (RL). It introduces a symbolic encoder to embedding vision and language features and extract structured primitives, utilizes a symbolic solver for data-efficient action sequencing, and leverages online RL to optimize generation via expansive exploration. Experiments on robotic manipulation benchmarks demonstrate that NS-VLA outperforms previous methods in both one-shot training and data-perturbed settings, while simultaneously exhibiting superior zero-shot generalizability, high data efficiency and expanded exploration space. Our code is available.

#### 1. 论文的核心问题与整体含义
视觉-语言-动作（VLA）模型旨在将自然语言指令转化为机器人的具体操作。然而，现有的端到端 VLA 模型（如 OpenVLA）面临三大挑战：
*   **缺乏结构化意识**：直接从视觉/语言回归到动作，忽略了机器人操作中可重用的“基元”（Primitives，如抓取、放置）之间的逻辑联系。
*   **数据依赖性强**：性能高度依赖大规模演示数据，在单样本（1-shot）或稀疏数据下表现极差。
*   **探索能力受限**：主要通过模仿学习（BC）训练，难以在专家演示之外的未知环境中进行自主探索。
**NS-VLA** 提出了一种神经符号（Neuro-Symbolic）框架，通过将连续的感知与离散的符号推理结合，并引入在线强化学习，旨在提升模型的数据效率、泛化能力和探索空间。

#### 2. 方法论：核心思想与关键技术
NS-VLA 采用层次化架构，将任务分解为“计划-推理-执行”三个阶段：
*   **神经符号编码器（Neuro-Symbolic Encoder）**：
    *   利用预训练 VLM（如 Qwen3-VL）将指令和初始图像转化为结构化的 JSON 计划（如 `[{"op": "pick", "args": "mug"}, ...]`）。
    *   引入**符号分类器**，在执行过程中实时预测当前处于哪个基元阶段，并施加“单调约束”，确保任务逻辑按顺序推进，避免动作反复跳变。
*   **符号求解器（Symbolic Solver）**：
    *   **视觉 Token 稀疏化**：基于当前基元（如“抓取杯子”），通过注意力机制只提取与目标物体相关的视觉特征，过滤背景噪声。
    *   **动作生成器**：使用轻量级 Transformer 接收稀疏视觉特征和基元嵌入，生成**动作块（Action Chunking）**，提高实时控制的稳定性。
*   **在线强化学习优化（Online RL）**：
    *   采用 **GRPO（组相对策略优化）** 算法，无需额外的价值函数网络。
    *   **奖励设计**：结合了任务成功奖励、基元完成里程碑奖励以及基于潜在状态的进度塑造奖励（Progress Shaping）。
    *   引入 KL 散度约束，防止在线探索时偏离预训练的模仿学习先验。

*   **训练策略创新**：将大语言模型中成熟的 GRPO 算法成功迁移至机器人动作控制领域，并配合进度奖励解决了稀疏奖励下的训练难题。
