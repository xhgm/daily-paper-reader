## 摘要
视觉-语言-动作（VLA）模型旨在根据视觉观测和自然语言指令控制机器人进行操作。然而，现有的分层和自回归范式通常会引入架构开销，面临时间不一致性和长时程误差累积的问题，且在不添加额外模块的情况下缺乏捕捉环境动力学的机制。为此，我们提出了 MMaDA-VLA，这是一个完全原生的预训练扩散 VLA 大模型，在单一框架中统一了多模态理解与生成。我们的核心思想是一种原生的离散扩散公式，它将语言、图像和连续机器人控制嵌入到一个离散的 token 空间中，并使用掩码 token 去噪训练单一骨干网络，以并行地联合生成未来目标观测和动作块（action chunk）。迭代去噪实现了全局、无序的细化，在提高长时程一致性的同时，无需辅助世界模型即可将动作植根于预测的未来视觉结果中。在仿真基准测试和真实世界任务中的实验表明，该模型达到了最先进的性能，在 LIBERO 上实现了 98.0% 的平均成功率，在 CALVIN 上实现了 4.78 的平均长度。

## Abstract
Vision-Language-Action (VLA) models aim to control robots for manipulation from visual observations and natural-language instructions. However, existing hierarchical and autoregressive paradigms often introduce architectural overhead, suffer from temporal inconsistency and long-horizon error accumulation, and lack a mechanism to capture environment dynamics without extra modules. To this end, we present MMaDA-VLA, a fully native pre-trained large diffusion VLA model that unifies multi-modal understanding and generation in a single framework. Our key idea is a native discrete diffusion formulation that embeds language, images, and continuous robot controls into one discrete token space and trains a single backbone with masked token denoising to jointly generate a future goal observation and an action chunk in parallel. Iterative denoising enables global, order-free refinement, improving long-horizon consistency while grounding actions in predicted future visual outcomes without auxiliary world models. Experiments across simulation benchmarks and real-world tasks show state-of-the-art performance, achieving 98.0% average success on LIBERO and 4.78 average length on CALVIN.

#### 1. 核心问题与研究动机
现有的视觉-语言-动作（VLA）模型在机器人操控任务中主要面临三大挑战：
*   **架构冗余与信息损失：** 许多模型采用分层设计（如在 VLM 上外挂策略头），增加了训练成本并可能导致模块间的信息保真度下降。
*   **长程一致性差：** 自回归（Autoregressive）生成方式在处理动作序列时，容易产生时间上的不一致性，并导致长程任务中的误差累积。
*   **缺乏动力学感知：** 现有范式通常缺乏显式的环境动力学建模，除非引入额外的世界模型模块。
*   **动作顺序偏置：** 动作空间的维度（如 7-DoF）本质上是无序的，强行使用自回归顺序会引入不必要的归纳偏置。

#### 2. 方法论：核心思想与技术细节
MMaDA-VLA 提出了一个**完全原生的预训练大型离散扩散 VLA 模型**，其核心在于将理解与生成统一在单一框架内：
*   **统一离散 Token 空间：** 使用统一的 Token 化策略。文本使用 LLaDA 分词器，图像使用 MAGVIT-v2 量化器，连续动作则被离散化为 256 个 bin。所有模态共享一个词表。
*   **原生离散扩散公式：** 采用掩码 Token 建模（Masked Token Modeling）。在训练时，随机掩盖目标观测图像和动作块的 Token，通过单一骨干网络进行去噪重构。
*   **并行生成机制：** 模型在推理时通过迭代去噪，**并行**生成未来的“目标观测图像”和“动作块（Action Chunk）”。这种设计让动作生成能够基于预测的未来视觉结果进行对齐，起到了隐式世界模型的作用。
*   **混合注意力机制（Hybrid Attention）：** 模态内部采用双向全注意力（Bidirectional Full Attention）以实现全局信息交换；模态之间采用因果注意力（Causal Attention）以确保信息流向的合理性。
*   **推理加速：** 引入了免训练的 KV 缓存（Key-Value Cache）框架，通过选择性刷新策略减少迭代去噪带来的延迟，满足实时控制需求。

#### 7. 优点与亮点
*   **架构优雅：** 实现了真正的“原生”统一，无需复杂的策略头或多阶段推理。
*   **无序性友好：** 离散扩散去噪完美契合了动作空间维度无序的特性，避免了自回归带来的顺序偏置。
*   **闭环纠错：** 实验观察到模型在真实世界中具有一定的错误恢复能力，能够根据视觉反馈修正动作。

*   **视觉生成精度有限：** 生成的未来目标图像在精细细节（如夹爪闭合状态、复杂物体纹理）上存在模糊，虽然足以引导动作，但视觉保真度仍有提升空间。
*   **推理开销：** 尽管有缓存加速，但迭代去噪的计算成本仍高于单次前向传播的自回归模型。
*   **数据依赖：** 模型性能高度依赖于大规模、高质量的跨化身（Cross-embodiment）预训练数据。
