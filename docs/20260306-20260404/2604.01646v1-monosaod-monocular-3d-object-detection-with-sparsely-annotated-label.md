## 摘要
单目3D目标检测在密集标注的数据集上已取得显著性能。然而，由于3D标注成本高昂，当仅有部分物体被标注时，其表现往往不尽如人意。这种稀疏标注的设置在现实场景中十分常见，因为对每个物体进行标注是不切实际的。为了解决这一问题，我们提出了一个用于稀疏标注单目3D目标检测的新型框架，包含两个核心模块。首先，我们提出了路面感知补丁增强（Road-Aware Patch Augmentation, RAPA），该模块通过将分割后的物体补丁增强到路面区域，同时保持3D几何一致性，从而充分利用稀疏标注。其次，我们提出了基于原型的过滤（Prototype-Based Filtering, PBF），通过原型相似度和深度不确定性过滤预测结果，从而生成高质量的伪标签。它维护全局2D RoI特征原型，并选择与学习到的原型特征一致且具有可靠深度估计的伪标签。我们的训练策略结合了保持几何特性的增强和原型引导的伪标签技术，以在稀疏监督下实现鲁棒的检测。大量实验证明了所提方法的有效性。源代码可在 https://github.com/VisualAIKHU/MonoSAOD 获取。

## Abstract
Monocular 3D object detection has achieved impressive performance on densely annotated datasets. However, it struggles when only a fraction of objects are labeled due to the high cost of 3D annotation. This sparsely annotated setting is common in real-world scenarios where annotating every object is impractical. To address this, we propose a novel framework for sparsely annotated monocular 3D object detection with two key modules. First, we propose Road-Aware Patch Augmentation (RAPA), which leverages sparse annotations by augmenting segmented object patches onto road regions while preserving 3D geometric consistency. Second, we propose Prototype-Based Filtering (PBF), which generates high-quality pseudo-labels by filtering predictions through prototype similarity and depth uncertainty. It maintains global 2D RoI feature prototypes and selects pseudo-labels that are both feature-consistent with learned prototypes and have reliable depth estimates. Our training strategy combines geometry-preserving augmentation with prototype-guided pseudo-labeling to achieve robust detection under sparse supervision. Extensive experiments demonstrate the effectiveness of the proposed method. The source code is available at https://github.com/VisualAIKHU/MonoSAOD .

#### 1. 论文的核心问题与整体含义（研究动机和背景）
单目3D目标检测（M3OD）旨在仅通过单张RGB图像恢复物体的空间位置和维度。尽管现有方法在密集标注的数据集上表现优异，但在现实场景中，获取精确的3D标注（深度、尺寸、朝向）成本极高（比2D标注慢3-16倍），导致实际数据往往呈现**稀疏标注（Sparsely Annotated）**状态。
*   **核心挑战**：现有的稀疏标注检测（SAOD）方法多针对2D任务（仅依赖分类置信度）或点云任务（依赖LiDAR深度），无法直接应用于缺乏显式几何监督的单目3D场景。
*   **研究动机**：在仅有少量物体被标注的情况下，如何通过增强数据多样性和生成高可靠性的3D伪标签来提升模型性能。

#### 2. 论文提出的方法论
MonoSAOD 框架采用**教师-学生（Teacher-Student）结构**，核心包含两个模块：

*   **路面感知补丁增强（RAPA）**：
    *   **核心思想**：将其他场景中已标注的物体“剪切-粘贴”到当前场景，但必须符合物理规律。
    *   **关键技术**：利用 SAM（Segment Anything Model）提取纯净的物体补丁；利用路面分割掩码确定合理的放置区域；通过**3D感知变换**，根据相机外参调整物体的3D中心、水平位移和旋转角度，确保增强后的物体在深度、比例和朝向上的几何一致性。
*   **基于原型的过滤（PBF）**：
    *   **核心思想**：联合考虑2D语义一致性和3D几何可靠性来筛选伪标签，而非仅依赖置信度。
    *   **关键技术**：
        *   **几何可靠性过滤**：利用拉普拉斯偶然不确定性（Aleatoric Uncertainty）估计深度预测的可靠性。
        *   **语义一致性过滤**：维护一个全局的类特定特征原型库（Prototypes），计算预测目标的 RoI 特征与原型的余弦相似度。
        *   **GT Bank 更新**：只有通过双重阈值筛选的预测才会被存入 GT Bank，作为后续训练的补充监督信号。

*   **数据集**：主要使用 **KITTI 3D Object Detection** 基准，并额外在 **Foggy KITTI**（雾天模拟）上测试鲁棒性。
*   **稀疏设置**：随机抽取 30%、50%、70% 的标注作为已知标签。
*   **Benchmark 与对比方法**：
    *   **基础检测器**：MonoDETR（主要）和 MonoDGP（验证泛化性）。
    *   **对比 SAOD 方法**：Co-mining、SparseDet、Calibrated Teacher、Co-student 等。
*   **评价指标**：采用 AP R40 评估 3D 检测（$AP_{3D}$）和鸟瞰图检测（$AP_{BEV}$）。

*   **实验规模**：论文涵盖了不同稀疏比例（10% 到 70%）的对比实验、针对 RAPA 和 PBF 各组件的消融实验、超参数敏感性分析。
*   **充分性评价**：
    *   **泛化性验证**：将方法作为插件应用于不同的检测器（MonoDETR 和 MonoDGP），均取得提升。
    *   **鲁棒性验证**：在恶劣天气（雾天）和存在分割噪声（掩码腐蚀/膨胀）的情况下进行了测试。

#### 6. 论文的主要结论与发现
*   **性能飞跃**：在 30% 稀疏标注下，MonoSAOD 显著优于现有 SAOD 方法。例如在 KITTI 验证集上，相比 Baseline 提升了约 10% 的 $AP_{3D}$。
*   **几何一致性的重要性**：实验证明，简单的 2D 复制粘贴会引入噪声，而保持 3D 几何一致性的 RAPA 模块对 M3OD 至关重要。
*   **置信度的局限性**：单纯的分类置信度不能反映深度预测的准确性，结合深度不确定性和特征原型的 PBF 模块能更有效地剔除错误的伪标签。

#### 7. 优点
*   **物理合理性**：RAPA 模块考虑了相机参数和路面约束，生成的增强图像比传统方法更真实。
*   **双重验证机制**：PBF 模块通过“语义+几何”双重过滤，解决了单目 3D 检测中伪标签质量难以评估的痛点。
*   **即插即用**：该框架不改变底层检测器结构，具有良好的通用性。

#### 8. 不足与局限
*   **性能差距**：尽管提升显著，但与 100% 全监督训练的性能相比仍有较大差距（尤其在极高稀疏度下）。
*   **依赖预处理**：RAPA 依赖于 SAM 分割和路面掩码的质量，虽然实验证明其对噪声有一定容忍度，但在极端复杂的非结构化道路上可能受限。
*   **类别局限**：实验主要集中在“汽车（Car）”类别，对于行人、骑行者等小尺寸或形变较大的目标，原型的稳定性有待进一步验证。
