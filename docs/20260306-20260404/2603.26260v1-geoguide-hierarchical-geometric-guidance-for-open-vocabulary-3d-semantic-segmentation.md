## 摘要
开放词汇 3D 语义分割旨在分割训练集之外的任意类别。现有方法主要依赖于从 2D 开放词汇模型中蒸馏知识。然而，将 3D 特征对齐到 2D 表示空间限制了内在的 3D 几何学习，并继承了 2D 预测中的误差。为了解决这些局限性，我们提出了 GeoGuide，这是一个利用预训练 3D 模型来整合层级几何-语义一致性的新型框架，用于开放词汇 3D 分割。具体而言，我们引入了一个基于不确定性的超点蒸馏模块，用于融合几何和语义特征以估计逐点不确定性，自适应地加权超点内的 2D 特征，在抑制噪声的同时保留判别性信息，从而增强局部语义一致性。此外，我们的实例级掩码重建模块利用几何先验，通过重建完整的实例掩码来强制执行实例内的语义一致性。另外，我们的实例间关系一致性模块对齐几何和语义相似度矩阵，以校准同类别物体间的跨实例一致性，从而缓解由视角引起的语义漂移。在 ScanNet v2、Matterport3D 和 nuScenes 上的广泛实验证明了 GeoGuide 的卓越性能。

## Abstract
Open-vocabulary 3D semantic segmentation aims to segment arbitrary categories beyond the training set. Existing methods predominantly rely on distilling knowledge from 2D open-vocabulary models. However, aligning 3D features to the 2D representation space restricts intrinsic 3D geometric learning and inherits errors from 2D predictions. To address these limitations, we propose GeoGuide, a novel framework that leverages pretrained 3D models to integrate hierarchical geometry-semantic consistency for open-vocabulary 3D segmentation. Specifically, we introduce an Uncertainty-based Superpoint Distillation module to fuse geometric and semantic features for estimating per-point uncertainty, adaptively weighting 2D features within superpoints to suppress noise while preserving discriminative information to enhance local semantic consistency. Furthermore, our Instance-level Mask Reconstruction module leverages geometric priors to enforce semantic consistency within instances by reconstructing complete instance masks. Additionally, our Inter-Instance Relation Consistency module aligns geometric and semantic similarity matrices to calibrate cross-instance consistency for same-category objects, mitigating viewpoint-induced semantic drift. Extensive experiments on ScanNet v2, Matterport3D, and nuScenes demonstrate the superior performance of GeoGuide.

### 1. 核心问题与研究背景
*   **核心问题**：现有的开放词汇 3D 语义分割方法过度依赖从 2D 预训练模型（如 CLIP）向 3D 模型的知识蒸馏。这种“2D 驱动”的模式存在三大缺陷：
    1.  **几何特征缺失**：强行将 3D 特征对齐到 2D 表示空间，忽略了 3D 空间固有的几何结构。
    2.  **误差继承**：2D 模型在处理遮挡、光照变化和边界模糊时容易产生错误预测，3D 模型会直接继承这些噪声。
    3.  **语义不一致**：由于视角变化，同一类别的物体在不同 2D 视图下的特征分布存在偏移，导致 3D 空间中语义破碎。
*   **研究动机**：利用预训练 3D 模型提供的几何先验来纠正 2D 蒸馏过程中的偏差，通过层级化的几何引导实现更鲁棒的语义对齐。

### 2. 方法论与关键技术
GeoGuide 框架引入了三个层级的几何-语义一致性模块，从局部到全局引导蒸馏过程：

*   **基于不确定性的超点蒸馏 (USD) —— 局部级**：
    *   **核心思想**：利用超点（几何同质区域）应具有相同语义的特性。
    *   **技术细节**：融合预训练 3D 特征与 2D 语义特征，通过一个轻量级 MLP 预测逐点**可靠性权重（Uncertainty Weights）**。在超点内部进行加权聚合，抑制噪声像素，增强局部语义的纯净度。
*   **实例级掩码重建 (IMR) —— 实例级**：
    *   **核心思想**：利用几何先验恢复被遮挡或不完整的实例区域。
    *   **技术细节**：采用类无关的 3D 实例分割生成提案，随机遮蔽实例的部分区域，要求模型利用剩余部分的特征重建完整的实例掩码。这强制了同一实例内部所有点在语义空间的高度一致性。
*   **实例间关系一致性 (IIRC) —— 全局级**：
    *   **核心思想**：几何相似的物体在语义上也应相似。
    *   **技术细节**：计算实例间（及超点间）的**几何相似度矩阵**和**语义相似度矩阵**。通过 MSE 损失对齐这两个矩阵，确保即使视角不同，同类物体的语义特征也能保持稳定，缓解语义漂移。

### 6. 主要结论与发现
*   **性能领先**：GeoGuide 在所有数据集上均达到 SOTA。在 ScanNet v2 上达到 64.8 mIoU，显著超过了之前的基准 SAS（+5.6 mIoU）。

*   **泛化能力强**：在跨域测试中，GeoGuide 表现出极小的性能衰减，证明其捕获的是领域无关的几何结构特征。

### 7. 优点与亮点
*   **层级化设计**：从超点到实例再到实例间关系，构建了从局部到全局的完整约束链。
*   **不确定性建模**：USD 模块能够自适应地过滤 2D 投影带来的噪声，解决了 2D-3D 对齐中的核心痛点。
*   **推理高效**：推理阶段不需要 2D 图像和复杂的几何引导模块，保持了纯 3D 模型的简洁性。

### 8. 不足与局限
*   **依赖外部提案**：IMR 模块依赖于类无关的 3D 实例分割算法（如 SAI3D），如果初始实例提案质量极差，可能会影响重建效果。

*   **2D 依赖的残余**：尽管引入了几何引导，但语义信息的源头仍是 2D 模型，若 2D 模型对某些罕见类别的识别完全错误，几何引导也难以凭空生成正确的语义。
