## 摘要
虽然最近的前馈三维重建模型通过在单次传递中联合推断稠密几何和相机位姿来加速三维重建，但它们对稠密注意力机制的依赖带来了二次方复杂度，形成了极高的计算瓶颈，严重限制了推理速度。为了解决这一问题，我们提出了 Speed3R，这是一种端到端可训练的模型，其灵感源自运动恢复结构（SfM）的核心原理：即稀疏的关键点集足以实现鲁棒的位姿估计。Speed3R 采用双分支注意力机制，其中压缩分支创建粗粒度的上下文先验以引导选择分支，而选择分支仅对最具信息量的图像 token 执行细粒度注意力计算。该策略模仿了传统关键点匹配的效率，在 1000 视图序列上实现了显著的 12.4 倍推理加速，同时在几何精度方面引入了极小且可控的权衡。在 VGGT 和 $π^3$ 骨干网络的标准基准测试中，我们的方法以极低的计算成本实现了高质量重建，为高效的大规模场景建模铺平了道路。

## Abstract
While recent feed-forward 3D reconstruction models accelerate 3D reconstruction by jointly inferring dense geometry and camera poses in a single pass, their reliance on dense attention imposes a quadratic complexity, creating a prohibitive computational bottleneck that severely limits inference speed. To resolve this, we introduce Speed3R, an end-to-end trainable model inspired by the core principle of Structure-from-Motion: that a sparse set of keypoints is sufficient for robust pose estimation. Speed3R features a dual-branch attention mechanism where a compression branch creates a coarse contextual prior to guide a selection branch, which performs fine-grained attention only on the most informative image tokens. This strategy mimics the efficiency of traditional keypoint matching, achieving a remarkable 12.4x inference speedup on 1000-view sequences, while introducing a minimal, controlled trade-off in geometric accuracy. Validated on standard benchmarks with both VGGT and $π^3$ backbones, our method delivers high-quality reconstructions at a fraction of computational cost, paving the way for efficient large-scale scene modeling.

#### 1. 核心问题与研究动机
*   **核心问题**：当前的前馈（Feed-forward）三维重建模型（如 VGGT, $\pi^3$）虽然能单次推断几何与位姿，但依赖**全注意力机制（Dense Attention）**。这种机制的计算复杂度随输入 Token 数量（视图数 × 图像分块数）呈二次方增长，导致在处理长序列（如 1000 帧）或高分辨率图像时出现严重的计算瓶颈。
*   **研究动机**：借鉴传统运动恢复结构（SfM）的原理——即一组稀疏的关键点足以实现鲁棒的位姿估计。作者旨在设计一种可训练的稀疏注意力机制，在大幅提升推理速度的同时，保持前馈模型的重建精度。

#### 2. 方法论
Speed3R 的核心是提出了**全局稀疏注意力（Global Sparse Attention, GSA）**模块，作为密集注意力的替代方案：
*   **双分支架构**：
    *   **压缩分支（Compression Branch）**：通过对图像 Token 进行空间下采样（如 $4 \times 4$ 平均池化），生成场景的粗略全局摘要。它计算一个低分辨率的注意力得分矩阵作为“引导”。
    *   **选择分支（Selection Branch）**：根据压缩分支提供的引导得分，利用 **Top-K 选择机制** 识别出最相关的全分辨率图像块。查询（Query）仅对这些选定的稀疏 Token 进行精细注意力计算。
*   **门控聚合（Gated Aggregation）**：使用可学习的门控机制动态融合粗粒度上下文和细粒度局部细节。
*   **特殊 Token 处理**：对于相机位姿等关键信息（Special Tokens），仍保留全注意力计算以确保全局一致性。
*   **高效内核实现**：开发了基于 **Triton** 的融合内核，将流式 Top-K 算法集成到 FlashAttention 流程中，避免了存储完整的得分矩阵，显著降低内存占用并提升速度。
*   **知识蒸馏训练**：采用预训练的密集模型（如原始 VGGT 或 $\pi^3$）作为教师模型，通过深度和位姿的蒸馏损失引导稀疏学生模型的训练。
