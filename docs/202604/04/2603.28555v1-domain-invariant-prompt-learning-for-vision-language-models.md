## 摘要
像 CLIP 这样的大型预训练视觉-语言模型通过在共享特征空间中对齐图像和文本，彻底改变了计算机视觉，并通过提示（prompting）实现了鲁棒的零样本迁移。软提示（Soft-prompting）方法，如上下文优化（CoOp），通过学习一组上下文向量，有效地使这些模型适应下游识别任务。然而，CoOp 缺乏处理未见分布中领域偏移（domain shifts）的显式机制。为此，我们提出了领域不变上下文优化（DiCoOp），这是 CoOp 的一种扩展，专门针对领域泛化（domain generalization）进行了优化。通过采用对抗训练方法，DiCoOp 强制模型学习领域不变的提示，同时保持分类的判别能力。实验结果表明，在跨越不同视觉领域的领域泛化任务中，DiCoOp 一贯优于 CoOp。

## Abstract
Large pre-trained vision-language models like CLIP have transformed computer vision by aligning images and text in a shared feature space, enabling robust zero-shot transfer via prompting. Soft-prompting, such as Context Optimization (CoOp), effectively adapts these models for downstream recognition tasks by learning a set of context vectors. However, CoOp lacks explicit mechanisms for handling domain shifts across unseen distributions. To address this, we propose Domain-invariant Context Optimization (DiCoOp), an extension of CoOp optimized for domain generalization. By employing an adversarial training approach, DiCoOp forces the model to learn domain-invariant prompts while preserving discriminative power for classification. Experimental results show that DiCoOp consistently surpasses CoOp in domain generalization tasks across diverse visual domains.

#### 1. 核心问题与研究背景
*   **核心问题**：预训练视觉-语言模型（如 CLIP）虽然在零样本迁移上表现出色，但通过软提示（Soft-prompting，如 CoOp）进行下游任务适配时，往往会过拟合于训练数据的特定分布。当面对未见过的领域偏移（Domain Shift）时，模型的泛化能力（Domain Generalization, DG）会显著下降。
*   **研究背景**：现有的提示学习方法（如 CoOp）主要关注如何通过优化上下文向量来提升分类精度，但缺乏显式的机制来处理领域间的不变性。本文旨在设计一种既能保持分类判别力，又能消除领域偏差的提示学习框架。

#### 2. 方法论
*   **核心思想**：借鉴领域对抗神经网络（DANN）的思想，将**对抗训练**引入提示学习。通过引入**梯度反转层（GRL）**，强制模型学习到的提示向量（Context Vectors）无法区分输入图像所属的领域，从而提取领域不变的特征。
*   **关键技术细节**：
    *   **DiCoOp 框架**：在训练过程中进行两次前向传播：
        1.  **分类分支（Class Pass）**：最小化分类损失（负对数似然），优化提示向量以识别类别。
        2.  **领域分支（Domain Pass）**：通过 GRL 最大化领域分类损失，使提示向量对领域信息变得“模糊”。
    *   **三种提示策略**：
        1.  **SCP（共享上下文提示）**：所有提示词共同参与分类和对抗任务。
        2.  **DFP（领域优先提示）**：将提示词分为两半，前半部分负责领域信息（训练时冻结分类词），后半部分负责类别信息。
        3.  **CFP（类别优先提示）**：与 DFP 顺序相反，前半部分为类别词，后半部分为领域词。
    *   **损失函数**：$L(v) = L_{cls}(v) - \lambda L_{dom}(v)$，其中 $\lambda$ 控制对抗训练的强度。
