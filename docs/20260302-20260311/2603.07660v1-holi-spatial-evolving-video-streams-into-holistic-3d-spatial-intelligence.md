## 摘要
空间智能的追求从根本上依赖于大规模、细粒度的 3D 数据。然而，现有方法主要通过从有限数量的人工标注数据集中生成问答（QA）对来构建空间理解基准，而不是系统地从原始网络数据中标注新的大规模 3D 场景。因此，它们的可扩展性受到严重限制，且模型性能进一步受到这些狭隘策划的数据集中固有领域差距的阻碍。在这项工作中，我们提出了 Holi-Spatial，这是第一个全自动、大规模、具有空间感知能力的多模态数据集，它利用所提出的数据策划流水线，在无需人工干预的情况下从原始视频输入中构建而成。Holi-Spatial 支持多级空间监督，涵盖从具有渲染深度图的几何精确 3D 高斯泼溅（3DGS）重建，到物体级和关系语义标注，以及相应的空间问答（QA）对。遵循原则性和系统性的流水线，我们进一步构建了 Holi-Spatial-4M，这是第一个大规模、高质量的 3D 语义数据集，包含 1.2 万个优化的 3DGS 场景、130 万个 2D 掩码、32 万个 3D 边界框、32 万个实例描述、120 万个 3D 定位实例以及 120 万个涵盖各种几何、关系和语义推理任务的空间 QA 对。Holi-Spatial 在数据策划质量方面表现出卓越的性能，在 ScanNet、ScanNet++ 和 DL3DV 等数据集上显著优于现有的前馈和逐场景优化方法。此外，使用该数据集在空间推理任务上对视觉语言模型（VLM）进行微调，也显著提升了模型性能。

## Abstract
The pursuit of spatial intelligence fundamentally relies on access to large-scale, fine-grained 3D data. However, existing approaches predominantly construct spatial understanding benchmarks by generating question-answer (QA) pairs from a limited number of manually annotated datasets, rather than systematically annotating new large-scale 3D scenes from raw web data. As a result, their scalability is severely constrained, and model performance is further hindered by domain gaps inherent in these narrowly curated datasets.   In this work, we propose Holi-Spatial, the first fully automated, large-scale, spatially-aware multimodal dataset, constructed from raw video inputs without human intervention, using the proposed data curation pipeline. Holi-Spatial supports multi-level spatial supervision, ranging from geometrically accurate 3D Gaussian Splatting (3DGS) reconstructions with rendered depth maps to object-level and relational semantic annotations, together with corresponding spatial Question-Answer (QA) pairs.   Following a principled and systematic pipeline, we further construct Holi-Spatial-4M, the first large-scale, high-quality 3D semantic dataset, containing 12K optimized 3DGS scenes, 1.3M 2D masks, 320K 3D bounding boxes, 320K instance captions, 1.2M 3D grounding instances, and 1.2M spatial QA pairs spanning diverse geometric, relational, and semantic reasoning tasks.   Holi-Spatial demonstrates exceptional performance in data curation quality, significantly outperforming existing feed-forward and per-scene optimized methods on datasets such as ScanNet, ScanNet++, and DL3DV. Furthermore, fine-tuning Vision-Language Models (VLMs) on spatial reasoning tasks using this dataset has also led to substantial improvements in model performance.

*   **整体含义**：本文提出了 **Holi-Spatial**，这是首个全自动化的 3D 空间数据策划管线。它能将原始视频流转化为包含几何结构、语义掩码、3D 边界框、实例描述及空间问答（QA）的综合性数据集，并据此发布了包含 400 万条标注的 **Holi-Spatial-4M** 数据集。

### 2. 方法论
Holi-Spatial 采用三阶段自动化管线：
1.  **几何优化 (Geometric Optimization)**：
    *   利用单目深度估计模型（Depth-Anything-V3）初始化，结合运动恢复结构（SfM）获取相机位姿。
    *   使用 **3D 高斯泼溅 (3DGS)** 进行逐场景优化，并引入几何正则化约束，以消除深度估计中的噪声和“漂浮物”，生成多视图一致的精确几何结构。
2.  **图像级感知 (Image-level Perception)**：
    *   在关键帧上利用 VLM（如 Gemini-Pro）生成开放词汇的类别标签，并维护一个“动态类别内存”以确保跨帧语义一致。
    *   利用 **SAM3** 生成高质量 2D 掩码，并通过渲染的深度图将像素反投影至 3D 空间，生成初始的 3D 定向边界框（OBB）。
3.  **场景级精炼 (Scene-level Refinement)**：
    *   **多视图合并**：通过 3D IoU 聚类合并来自不同视角的重复物体观测。
    *   **重力对齐**：检测地面平面，校正物体的旋转角度（Roll/Pitch）。
    *   **置信度过滤与 Agent 验证**：采用三级决策机制。高置信度直接保留，低置信度剔除；处于中间地带的样本由一个 **VLM Agent**（配备缩放和重分割工具）进行二次人工式核验。
    *   **QA 生成**：基于验证后的 3D 实例，自动合成涵盖相机中心和物体中心任务的 125 万个空间 QA 对。

### 3. 实验设计
*   **数据集与场景**：在 **ScanNet**、**ScanNet++** 和 **DL3DV-10K** 三个主流数据集上验证管线质量。
*   **Benchmark**：
    *   **3D 空间理解**：评估深度估计（F1）、2D 分割（IoU）和 3D 检测（AP25/50）。

*   **训练细节**：微调 Qwen3-VL 系列模型（2B 和 8B 版本），采用 1024 的全局 Batch Size，训练时长为 1 个 Epoch。
*   **数据规模**：处理了 1.2 万个 3DGS 场景，生成了 400 万级标注。

*   **实验规模**：论文进行了大规模的定量评估，涵盖了从底层几何（深度）到高层语义（检测、QA）的多个维度。

*   **客观性**：为了公平评估，研究者对部分场景进行了人工重新标注作为“金标准”GT，对比结果显示 Holi-Spatial 的自动标注在边界清晰度和类别准确性上甚至优于部分原始数据集的人工标签。实验设计较为客观、充分。

### 6. 主要结论与发现
*   **标注质量飞跃**：Holi-Spatial 在 ScanNet++ 上的 3D 检测 AP50 达到了 70.05，远超现有 3D-VLM 方法（如 LLaVA-3D 仅为 4.80）。

### 7. 优点
*   **全自动化与可扩展性**：彻底摆脱人工，能够将互联网上几乎无限的视频资源转化为训练数据。
*   **高保真几何**：结合 3DGS 和深度先验，解决了传统自动标注中 3D 边界框不准、漂浮物多的痛点。
*   **闭环验证**：引入 VLM Agent 作为“虚拟标注员”，显著提高了长尾物体的召回率和准确性。

*   **计算开销**：逐场景的 3DGS 优化和 VLM Agent 的频繁调用需要巨大的计算资源，难以实现实时标注。
