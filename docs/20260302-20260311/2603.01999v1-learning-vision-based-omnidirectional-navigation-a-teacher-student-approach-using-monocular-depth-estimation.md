## 摘要
工业环境中的可靠避障需要 3D 场景理解，但广泛使用的 2D 激光雷达（LiDAR）传感器仅能感知环境的单个水平切面，从而遗漏了扫描平面上方或下方的关键障碍物。我们提出了一种用于基于视觉的移动机器人导航的教师-学生框架，该框架消除了对激光雷达传感器的需求。在 NVIDIA Isaac Lab 中通过近端策略优化（PPO）训练的教师策略利用了考虑完整机器人足迹的特权 2D 激光雷达观测，以学习鲁棒的导航。学习到的行为被蒸馏到一个学生策略中，该策略仅依赖于由微调后的 Depth Anything V2 模型从四个 RGB 摄像头预测的单目深度图。完整的推理流水线包括单目深度估计（MDE）、策略执行和电机控制，完全在安装于 DJI RoboMaster 平台上的 NVIDIA Jetson Orin AGX 上运行，推理过程无需外部计算。在仿真中，学生策略实现了 82-96.5% 的成功率，始终优于标准的 2D 激光雷达教师策略（50-89%）。在真实世界实验中，当在具有复杂 3D 几何形状的障碍物（如悬空结构和低矮物体）周围导航时，基于 MDE 的学生策略优于 2D 激光雷达教师策略，因为这些障碍物超出了 2D 激光雷达的单个扫描平面。

## Abstract
Reliable obstacle avoidance in industrial settings demands 3D scene understanding, but widely used 2D LiDAR sensors perceive only a single horizontal slice of the environment, missing critical obstacles above or below the scan plane. We present a teacher-student framework for vision-based mobile robot navigation that eliminates the need for LiDAR sensors. A teacher policy trained via Proximal Policy Optimization (PPO) in NVIDIA Isaac Lab leverages privileged 2D LiDAR observations that account for the full robot footprint to learn robust navigation. The learned behavior is distilled into a student policy that relies solely on monocular depth maps predicted by a fine-tuned Depth Anything V2 model from four RGB cameras. The complete inference pipeline, comprising monocular depth estimation (MDE), policy execution, and motor control, runs entirely onboard an NVIDIA Jetson Orin AGX mounted on a DJI RoboMaster platform, requiring no external computation for inference. In simulation, the student achieves success rates of 82-96.5%, consistently outperforming the standard 2D LiDAR teacher (50-89%). In real-world experiments, the MDE-based student outperforms the 2D LiDAR teacher when navigating around obstacles with complex 3D geometries, such as overhanging structures and low-profile objects, that fall outside the single scan plane of a 2D LiDAR.

*   **研究背景**：虽然 3D 传感器（如双目相机或 ToF）可以解决此问题，但其成本、重量和集成复杂度较高。本文旨在利用**单目深度估计（MDE）**技术，将廉价的 RGB 摄像头转化为深度传感器，并通过强化学习实现全向视觉导航。

### 2. 方法论
*   **核心思想**：采用**教师-学生（Teacher-Student）框架**。先在仿真中训练一个拥有“特权信息”的教师策略，再将其知识蒸馏给仅依赖视觉深度的学生策略。

    *   **单目深度估计（MDE）**：选用 **Depth Anything V2 (ViT-Small)** 模型。为了减小仿真与现实的差距，研究者在真实的仓库环境（2.4万对图像）中对其进行了微调。
    *   **学生策略（行为蒸馏）**：学生策略输入为 4 路 RGB 相机生成的深度图。通过**行为克隆（Behavior Cloning）**学习教师的动作输出。
    *   **噪声增强流水线**：在训练学生时，对仿真深度图施加 7 阶段增强（包括高斯模糊、运动模糊、弹性变形、尺度偏移等），以模拟真实 MDE 输出的伪影。
*   **算法流程**：
    1.  **Step I**：在线训练教师策略（基于特权 LiDAR）。
    2.  **Step II**：运行教师策略并记录数据（深度图 + 动作）。
    3.  **Step III**：离线监督学习训练学生策略（基于噪声增强深度图）。

### 6. 主要结论与发现
*   **视觉方案更优**：在仿真中，MDE 学生策略的成功率（82%-96.5%）始终高于标准 2D LiDAR 教师（50%-89%）。

### 7. 优点
*   **低成本高感知**：用廉价 RGB 相机实现了超越昂贵 2D LiDAR 的 3D 避障效果。
*   **Sim-to-Real 迁移成功**：通过特权学习和深度图噪声增强，成功将仿真策略迁移至真实机器人，无需在真实世界进行 RL 训练。
*   **全向移动支持**：针对麦克纳姆轮底盘设计，实现了灵活的全向避障。

### 8. 不足与局限
*   **静态环境限制**：实验仅在静态障碍物环境下进行，未验证在行人、叉车等动态环境中的表现。
*   **运动震荡**：实测轨迹显示学生策略存在一定的震荡现象，可能源于推理延迟或 MDE 的尺度不稳定性。
*   **远距离精度**：MDE 模型在远距离存在尺度漂移（Scale Mismatch），导致远距离避障精度下降。
*   **缺乏外部对比**：未与传统的基于 3D 视觉（如双目 SLAM 或代价图）的成熟导航框架进行横向对比。
